Detailed Description

A trigger that is triggered by a specified event and fires after a delay.

.. code-block:: xml

<Grid xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors"> <Button Content="Close"> <b:Interaction.Triggers> <b:TimerTrigger EventName="Click" MillisecondsPerTick="500" TotalTicks="1"> <b:InvokeCommandAction Command="{Binding CloseCommand}"/> </b:TimerTrigger> </b:Interaction.Triggers> </Button> </Grid>

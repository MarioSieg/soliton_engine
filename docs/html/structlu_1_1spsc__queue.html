Detailed Description

template<typename T, typename Alloc = std::allocator<T>>
struct lu::spsc_queue< T, Alloc >

Bounded single-producer single-consumer wait and lock-free queue. Only a single writer thread can perform enqueue operations and only a single reader thread can perform dequeue operations. Any other usage is invalid.

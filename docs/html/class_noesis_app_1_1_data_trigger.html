Detailed Description

Represents a trigger that performs actions when the bound data meets a specified condition.

.. code-block:: xml

<Grid xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors"> <TextBlock Text="{Binding Name}"> <b:Interaction.Triggers> <b:DataTrigger Binding="{Binding Quantity}" Comparison="GreaterThan" Value="25"> <b:ChangePropertyAction PropertyName="FontSize" Value="21" /> </b:DataTrigger> </b:Interaction.Triggers> </TextBlock> </Grid>

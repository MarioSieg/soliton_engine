Detailed Description

A trigger that listens for a specified event on its source and fires when that event is raised.

.. code-block:: xml

<Grid xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors"> <TextBlock Text="{Binding Name}"> <b:Interaction.Triggers> <b:EventTrigger EventName="MouseEnter"> <b:ChangePropertyAction PropertyName="FontWeight" Value="Bold"/> </b:EventTrigger> </b:Interaction.Triggers> </TextBlock> </Grid>

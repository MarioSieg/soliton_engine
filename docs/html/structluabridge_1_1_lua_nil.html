Detailed Description

Type tag for representing LUA_TNIL.

Construct one of these using LuaNil () to represent a Lua nil. This is faster than creating a reference in the registry to nil. Example:

LuaRef t (LuaRef::createTable (L));
...
t ["k"] = LuaNil (); // assign nil

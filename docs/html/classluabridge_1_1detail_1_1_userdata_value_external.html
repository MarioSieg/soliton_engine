Detailed Description

template<class T>
class luabridge::detail::UserdataValueExternal< T >

Wraps an external value type to a class object inside a Lua userdata.

The lifetime of the object is managed by Lua. The object is constructed inside the userdata using an already constructed object provided externally, and it is destructed by a deallocator function provided.

Detailed Description

Constructor & Destructor Documentation

◆ TableItem() [1/2]

luabridge::LuaRef::TableItem::TableItem ( lua_State * L,
int tableRef )
inline

Construct a TableItem from a table value.

The table is in the registry, and the key is at the top of the stack. The key is popped off the stack.

Parameters
LA lua state.
tableRefThe index of a table in the Lua registry.

◆ TableItem() [2/2]

luabridge::LuaRef::TableItem::TableItem ( const TableItem & other)
inline

Create a TableItem via copy constructor.

It is best to avoid code paths that invoke this, because it creates an extra temporary Lua reference. Typically this is done by passing the TableItem parameter as a const reference.

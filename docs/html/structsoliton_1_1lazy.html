Detailed Description

template<typename T>
struct soliton::lazy< T >

Container for lazy initialization. Construct using the .New(args) method. Object is destroyed in the destructor like normally.

Template Parameters
TThe type. Must be conforming to Option<T>. (C++ 17 destructible requirements N4828)

Detailed Description

template<class T, T... Values>
struct luabridge::Enum< T, Values >

LuaBridge enum wrapper for enums as integers.

An enum exposed with this class will just be decayed to lua as integer. It's responsibility of the developer to make sure that a lua integer could be converted back to C++. Failing to validate a lua integer before converting to the corresponding C++ enum value could lead to a C++ enum that has no defined value.

For improved security, specify which values the enum will have, so runtime validation could be performed.

Detailed Description

A Trigger that is triggered by a specified gamepad button.

.. code-block:: xml

<UserControl xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:b="http://schemas.microsoft.com/xaml/behaviors" xmlns:noesis="clr-namespace:NoesisGUIExtensions;assembly=Noesis.GUI.Extensions"> <b:Interaction.Triggers> <noesis:GamepadTrigger FiredOn="ButtonDown" Button="Cancel"> <b:InvokeCommandAction Command="{Binding ExitCommand}" /> </noesis:GamepadTrigger> </b:Interaction.Triggers> ... </UserControl>

◆ GetActiveOnFocus()

bool GamepadTrigger::GetActiveOnFocus ( ) const

If true, the Trigger listens to ButtonDown or ButtonUp events raised by its Source object, which means that element must have focus for the trigger to fire. If false, the Trigger listens at the root, so any unhandled ButtonDown and ButtonUp messages will be caught

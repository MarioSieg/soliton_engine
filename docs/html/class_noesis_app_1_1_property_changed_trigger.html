Detailed Description

Represents a trigger that performs actions when the bound data changes.

.. code-block:: xml

<Grid xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors"> <Slider x:Name="Percentage" Minimum="0" Maximum="100"> <b:Interaction.Triggers> <b:PropertyChangedTrigger Binding="{Binding Value, ElementName=Percentage}"> <b:PlaySoundAction Source="Sounds/SliderDrag.wav"/> </b:PropertyChangedTrigger> </b:Interaction.Triggers> </Slider> </Grid>

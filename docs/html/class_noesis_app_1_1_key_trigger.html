Detailed Description

A Trigger that is triggered by a keyboard event. If Key and Modifiers are detected, it fires.

.. code-block:: xml

<UserControl xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors"> <b:Interaction.Triggers> <b:KeyTrigger FiredOn="KeyDown" Key="Escape"> <b:InvokeCommandAction Command="{Binding ExitCommand}" /> </b:KeyTrigger> </b:Interaction.Triggers> ... </UserControl>

◆ GetActiveOnFocus()

bool KeyTrigger::GetActiveOnFocus ( ) const

If true, the Trigger only listens to its trigger Source object, which means that element must have focus for the trigger to fire. If false, the Trigger listens at the root, so any unhandled KeyDown/Up messages will be caught

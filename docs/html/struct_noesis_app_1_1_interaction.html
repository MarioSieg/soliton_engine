Detailed Description

Manages a collection of behaviors and triggers that expand the object functionality from XAML.

.. code-block:: xml

<UserControl xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors"> <b:Interaction.Behaviors> <b:MouseDragElementBehavior /> <b:Interaction.Behaviors> <b:Interaction.Triggers> <b:EventTrigger EventName="Loaded"> <b:GoToStateAction StateName="Init" /> </b:EventTrigger> <b:Interaction.Triggers> </UserControl>

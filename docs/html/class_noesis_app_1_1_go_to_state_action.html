Detailed Description

An action that will transition a FrameworkElement to a specified VisualState when invoked.

.. code-block:: xml

<UserControl xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors"> <b:Interaction.Triggers> <b:PropertyChangedTrigger Binding="{Binding State}"> <b:GoToStateAction StateName="{Binding State}" /> </b:PropertyChangedTrigger> </b:Interaction.Triggers> ... </UserControl>

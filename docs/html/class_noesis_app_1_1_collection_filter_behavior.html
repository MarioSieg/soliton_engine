Detailed Description

A behavior that filters a collection depending on the supplied Predicate object.

.. code-block:: xml

<Grid xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors" xmlns:noesis="clr-namespace:NoesisGUIExtensions;assembly=Noesis.GUI.Extensions"> <b:Interaction.Behaviors> <noesis:CollectionFilterBehavior x:Name="FilterBehavior" ItemsSource="{Binding Books}" Predicate="{Binding TextPredicate}" /> </b:Interaction.Behaviors> <TextBox Text="{Binding SearchText}"> <ListBox ItemsSource="{Binding FilteredItems, ElementName=FilterBehavior}" > </Grid>

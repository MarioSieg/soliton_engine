Detailed Description

A behavior that sorts a collection depending on the supplied Comparer object.

.. code-block:: xml

<Grid xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation" xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml" xmlns:b="http://schemas.microsoft.com/xaml/behaviors" xmlns:noesis="clr-namespace:NoesisGUIExtensions;assembly=Noesis.GUI.Extensions"> <b:Interaction.Behaviors> <noesis:CollectionSortBehavior x:Name="SortBehavior" ItemsSource="{Binding Books}" Comparer="{Binding BookComparer}" /> </b:Interaction.Behaviors> <ListBox ItemsSource="{Binding SortedItems, ElementName=SortBehavior}" > </Grid>

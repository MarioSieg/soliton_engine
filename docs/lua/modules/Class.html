XmlParser.new (_handler, _options)
Instantiates a XmlParser object.

Parameters:

  • _handler Handler module to be used to convert the XML string to another formats. See the available handlers at the handler directory. Usually you get an instance to a handler module using, for instance: local handler = require("xmlhandler/tree").
  • _options Options for this XmlParser instance.

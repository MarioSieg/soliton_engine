xml2lua.parser (handler)
Instantiates a XmlParser object to parse a XML string

Parameters:

  • handler Handler module to be used to convert the XML string to another formats. See the available handlers at the handler directory. Usually you get an instance to a handler module using, for instance: local handler = require("xmlhandler/tree").

Returns:

    a XmlParser object used to parse the XML

xml2lua.toXml (tb, tableName, level)
Converts a Lua table to a XML String representation.

Parameters:

  • tb Table to be converted to XML
  • tableName Name of the table variable given to this function, to be used as the root tag. If a value is not provided no root tag will be created.
  • level Only used internally, when the function is called recursively to print indentation

Returns:

    a String representing the table content in XML

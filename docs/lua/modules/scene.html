Scene.new (name:)
Creates a new, empty scene with the given name and makes it the active scene.

Parameters:

  • name: string, name of the new scene
Scene.load (file:, import_scale:, import_flags:)
Loads a scene from a given file and makes it the active scene. The file can be a 3d mesh file like .gltf, .fbx, .obj etc.. or a .lunam file. Importing an external file which is not .lunam will take much longer, because conversion is required.

Parameters:

  • file: string, path to the file to load
  • import_scale: number, scale factor to apply to the imported scene. Only applies if the scene is not a .lunam file
  • import_flags: number, flags to control the import process. Only applies if the scene is not a .lunam file
